# CUJ 02: Adding a Food Item (Manual)

## Goal
Manually log a meal with a specific name and calorie count.

## Prerequisites
- User is on the Dashboard.

## Steps
1. Click the Floating Action Button (Plus icon) at the bottom of the screen.
2. In the "Add Meal" drawer, select the first input field ("What did you eat?").
3. Enter a description (e.g., "Apple").
4. Select the second input field ("Calories").
5. Enter a numeric value (e.g., "95").
6. Click the "Add Meal" button (which enables only after both fields are filled).

## Expected Outcome
- The drawer closes.
- The new item "Apple" (95 cal) appears at the bottom of the "Today's Meals" list.
- The "Daily Balance" progress bar updates to reflect the consumed calories.
- The "Cal left" countdown decreases by the entered amount.
