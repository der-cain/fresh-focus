# CUJ 01: Try Demo Mode

## Goal
Access the application dashboard and features without creating a permanent account.

## Prerequisites
- User is on the Landing/Login page.

## Steps
1. Navigate to the application root.
2. Locate the "Try Demo Mode" button in the Hero section.
3. Click "Try Demo Mode".

## Expected Outcome
- The user is redirected to `/dashboard`.
- A temporary "demo" user session is initiated.
- The Dashboard display today's (empty) calorie log and the "Daily Balance" progress bar.
- If it's a new day/session, a Streak Reset or Streak Check modal may appear.
